
安装软件

## 1. 安装vim

（1）下载.vimrc文件

（2）安装Vundle插件：git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/Vundle.vim

（3）在vim中执行:PluginInstall命令

（4）YouCompleteMe问题：

unavailable: requires Vim 7.4.1578+：

      安装vim8.0

找不到python库：

      更新cmake到3.0版本

最后编译：

      $ cd ~/.vim/bundle/YouCompleteMe
      
      $ ./install.py (cmake required)
